**Title**
# U.S. Demographic & Income Analysis for Business Expansion

**Overview**
This project analyzes American Community Survey (ACS) Census data
to identify high-opportunity U.S. markets for business expansion.

**Tools Used**
• Python (Pandas)
• MySQL
• Power BI
• ACS Census Data

**Business Questions**
• Which states have strong purchasing power?
• Which regions show workforce strength?
• Where is education concentration highest?
• Which markets balance affordability and opportunity?

**Key Features**
• Data cleaning & transformation
• SQL business analysis
• Opportunity Score modeling
• Interactive Power BI dashboards
• Expansion market recommendations

**Dashboard Pages**
1. Introduction
2. Executive Summary
3. Workforce & Education Analysis
4. Expansion Recommendation

**Results**
The analysis identified states with strong income,
employment, and education indicators as high-potential
expansion markets.
